# 🤖 Eitaa Automation Dashboard

سیستم اتوماسیون ارسال پیام در ایتا با استفاده از:
- Node.js
- Playwright
- Express
- Socket.io
- Tailwind CSS (لوکال)

## ✨ امکانات
- داشبورد وب زنده
- لاگ Real-time
- کنترل Start / Stop
- ارسال خودکار پیام
- مدیریت Session
- پشتیبانی از محدودسازی ادمین‌ها

## 🚀 نصب و اجرا

### پیش‌نیازها
- Node.js v20+
- Git

```bash
git clone https://github.com/YOUR_USERNAME/eitaa-automation.git
